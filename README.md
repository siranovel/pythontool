## Requirement ##

## Usage ##
* pyimpinfo  
  $ pyimpinfo  
* pyosinfo  
  $ pyosinfo
* pysiteinfo  
  $ pysiteinfo
* pysysconfiginfo  
  $ pysysconfiginfo
* pysysinfo  
  $ pysysinfo
* pytarinfo  
  $ pytarinfo tarfile
* pypilinfo  
  $ pyimage 画像ファイル名
* pyrpminfo  
  $ pyrpminfo rpmファイル
* pyrpmtaginfo  
  $ pyrpmtaginfo
* pylz  
  $ pylz2  
  $ pylz3  
* pypass  
  $ pyuser

## install ##
    git clone https://github.com/siranovel/pythontool.git  
    cd pythontool  
    rpm -ivh <rpmファイル名>
